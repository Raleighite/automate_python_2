# Automate the Boring Stuff with Python — Part 2

Continuation of exercises from *Automate the Boring Stuff with Python* by Al Sweigart.

## Contents

- `guessTheNumber.py` — Number guessing game
- `rpsGame.py` — Rock paper scissors
- `vampire.py` / `vampire2.py` — Vampire number checker
- `magic8Ball.py` — Magic 8-ball simulator
- And more...

## Running

```bash
python3 <filename>.py
```

Requires Python 3.x. No external dependencies.
